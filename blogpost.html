Git y GitHub

Git es un software de Control de Versiones, el cual esta diseñado para ser eficiente y confiable, mantiene un historial de cambios, es perfecto para proyectos de desarrollo de Software.

GitHub es su lugar es un Repositorio Remoto, permite compartir el proyecto con otras personas en un equipo de trabajo o inclusive puede ser tomada como tu base personalizada de proyectos que puede ser tomada como tu CV.
